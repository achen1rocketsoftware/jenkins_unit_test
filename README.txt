   INTRODUCTION
   ============

   This set of examples demonstrates some of the capabilities of
   Micro Focus Unit Testing Framework.

   See the topic "Micro Focus Unit Testing Framework" in the Documentation
   included with this product (Eclipse) or online at the following url:

   http://www.microfocus.com/docs/links.asp?vc=mf_unit_testing

   =======

   Import the project into the workspace and run the Unit Tests.
   
   1. File -> Import -> Existing project into workspace
   
   2. Run -> Run As -> COBOL Unit Test
